SignPath Maven Plugin

Maven plugin that signs artifacts via the SignPath REST API.

Goals Overview

The plugin provides 1 goal to sign Maven build artifacts:

  • signpath:sign submits build artifacts to the SignPath REST API for signing and downloads the signed files once the signing request completes.

Usage

General instructions on how to use the SignPath Maven Plugin can be found on the usage page.

If you still have questions about how to use the plugin, have a look at the GitHub Discussions or open an issue.
